01 / Introduction

What is a Circle in Computer Graphics?

A circle is defined by its center point and radius. Unlike lines, circles require special consideration due to their curved nature.

Mathematical Definition

A circle is the set of all points in a plane that are at a fixed distance (radius) from a fixed point (center). The mathematical equation is:

(x - xc)² + (y - yc)² = r²
Circle equation with center (xc, yc) and radius r

Key Components

  • C
    Center (xc, yc): The fixed point from which all circle points are equidistant
  • r
    Radius (r): The distance from center to any point on the circle
  • P
    Point (x, y): Any point on the circle satisfies the equation

The equation can be rewritten to calculate y for each x: y = yc ± √(r² - (x - xc)²), but this involves expensive square root operations.

03 / Symmetry

8-Way Circle Symmetry

A circle exhibits perfect symmetry. By calculating one point, we can derive seven additional points through reflection, reducing computation by 8 times.

SYMMETRY_DEMO
Original Point (x, y)
Symmetric Points 8

How Symmetry Works

A circle centered at the origin is symmetric across both axes and all diagonal lines passing through the center. This means calculating one point in a single octant (1/8 of the circle) generates all other points.

Octant 1
(x, y)
Octant 2
(y, x)
Octant 3
(-y, x)
Octant 4
(-x, y)
Octant 5
(-x, -y)
Octant 6
(-y, -x)
Octant 7
(y, -x)
Octant 8
(x, -y)

Key Insight

When the circle is centered at an arbitrary point (xc, yc), the transformations become:

(xc + x, yc + y) | (xc + y, yc + x)
(xc - y, yc + x) | (xc - x, yc + y)
(xc - x, yc - y) | (xc - y, yc - x)
(xc + y, yc - x) | (xc + x, yc - y)

Why Symmetry Matters

By exploiting 8-way symmetry, circle drawing algorithms achieve an 8x performance improvement by computing only 1/8 of the circle.

Performance Impact

For a circle with radius r, approximately 2πr pixels need to be drawn. With symmetry, we only need to calculate points in one octant.

Total points ≈ 2πr ≈ 6.28r
Octant points ≈ r/√2 ≈ 0.707r
Reduction = 8x faster

06 / Next Steps

Circle Drawing Algorithms

Now that you understand circle symmetry, you're ready to learn the actual algorithms that use this property to draw circles efficiently.

How Algorithms Use Symmetry

Both Midpoint and Bresenham circle algorithms work by:

  • 1
    Calculate: Find points in only the first octant (0° to 45°)
  • 2
    Reflect: Apply 8-way symmetry transformations
  • 3
    Optimize: Use decision parameters to avoid floating-point math
plotCirclePoints(xc, yc, x, y) {
  draw(xc+x, yc+y); draw(xc+y, yc+x);
  draw(xc-y, yc+x); draw(xc-x, yc+y);
  draw(xc-x, yc-y); draw(xc-y, yc-x);
  draw(xc+y, yc-x); draw(xc+x, yc-y);
}
