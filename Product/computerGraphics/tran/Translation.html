01 / Definition

What is Translation?

Translation is a transformation that moves every point of an object by the same distance in a given direction. It shifts the object without altering its shape or orientation.

TRANSLATION_VISUALIZATION

Definition

Translation moves an object from its original position to a new position by adding translation values to each coordinate:

P'(x', y') = P(x, y) + T(tx, ty)
Translation operation
  • T
    Rigid Transformation: Preserves distances and angles
  • S
    Shape Preserved: No change in size or orientation
  • P
    Parallel Movement: All points move the same distance
02 / Mathematics

Translation Formula

The mathematical representation of translation uses simple addition to calculate new coordinates from original positions.

Coordinate Transformation

For any point P(x, y), the translated point P'(x', y') is calculated by:

x' = x + tx
X-coordinate translation
y' = y + ty
Y-coordinate translation

Parameters

  • tx
    Translation X: Distance to move along the X-axis (positive = right, negative = left)
  • ty
    Translation Y: Distance to move along the Y-axis (positive = down, negative = up)

04 / Vector Concept

Translation as a Vector

Translation can be represented as a vector that describes both the magnitude and direction of the movement.

Vector Representation

A translation vector T = (tx, ty) completely describes the transformation:

T = (tx, ty)
Translation vector notation

Vector Properties

  • M
    Magnitude: |T| = sqrt(tx^2 + ty^2) - the distance moved
  • D
    Direction: Angle from positive X-axis
  • A
    Additive: Multiple translations can be combined by adding vectors
T1 + T2 = (tx1 + tx2, ty1 + ty2)
Combining translations

05 / Applications

Real-World Applications

Translation is one of the most frequently used transformations in computer graphics, games, and user interfaces.

Common Uses

  • G
    Game Development: Moving characters, objects, and projectiles across the game world
  • A
    Animation: Creating smooth motion by applying incremental translations frame by frame
  • C
    Camera Movement: Panning the viewport to follow players or reveal content
  • U
    UI Positioning: Placing buttons, windows, and elements at specific screen coordinates
