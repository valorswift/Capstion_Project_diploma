01 / Definition

What is Scaling?

Scaling is a transformation that changes the size of an object by multiplying its coordinates with scale factors, while maintaining its shape proportions.

SCALING_VISUALIZATION

Definition

Scaling multiplies each coordinate of an object by a scale factor. This transformation can enlarge or shrink an object while preserving its basic shape.

P' = S × P
Scaling operation
  • sx
    Scale X: Horizontal scaling factor
  • sy
    Scale Y: Vertical scaling factor
  • O
    Origin: Scaling is performed relative to the origin
02 / Mathematics

Scaling Formula

The mathematical representation of scaling transformation using coordinate multiplication.

Coordinate Transformation

Each point (x, y) is transformed to a new position (x', y') by multiplying with the respective scale factors:

x' = x × sx
y' = y × sy
Scaling equations

Scale Factor Values

  • >1
    Enlargement: Object grows larger (s > 1)
  • =1
    No Change: Object stays the same (s = 1)
  • <1
    Reduction: Object shrinks (0 < s < 1)
  • -
    Reflection: Negative values mirror the object

04 / Scaling Types

Uniform vs Non-Uniform Scaling

Understanding the difference between proportional and disproportional scaling.

Comparison of Scaling Types

UNIFORM SCALING

When sx = sy, the object scales proportionally. The shape remains similar, only the size changes. Aspect ratio is preserved.

NON-UNIFORM SCALING

When sx ≠ sy, the object stretches or compresses. The shape changes, creating distortion in one direction.

Uniform Scaling

When both scale factors are equal (sx = sy = s), the object maintains its proportions. This is commonly used for zooming and resizing images without distortion.

x' = x × s
y' = y × s
Uniform scaling (sx = sy = s)

Non-Uniform Scaling

When scale factors differ (sx ≠ sy), the object stretches or compresses along one axis. This creates distortion but is useful for special effects and transformations.

x' = x × sx
y' = y × sy
Non-uniform scaling (sx ≠ sy)

06 / Matrix Form

Matrix Representation

Scaling can be represented as a transformation matrix in homogeneous coordinates.

MATRIX_VISUALIZATION

Scaling Matrix

In homogeneous coordinates, scaling is represented as a 3×3 matrix that can be combined with other transformations:

[x'] [sx 0 0] [x]
[y'] = [0 sy 0] [y]
[1 ] [0 0 1] [1]
Scaling matrix in homogeneous coordinates
  • M
    Matrix Form: Enables transformation composition
  • C
    Combination: Multiple scalings multiply together
  • I
    Identity: Scale factors of 1 produce no change
