01 / Introduction

What is Rotation?

Rotation is a geometric transformation that turns an object around a fixed point, changing its orientation while preserving its shape and size.

ROTATION_CONCEPT
Current Angle 0°
Rotation Center Origin (0, 0)
Direction Counter-clockwise

Definition

Rotation transforms each point of an object by moving it along a circular path around a fixed center point. The distance from the center remains constant.

θ (theta) = rotation angle
The angle determines how far to rotate
  • 1
    Fixed Point: All points rotate around this center
  • 2
    Angle θ: Determines the amount of rotation
  • 3
    Preservation: Shape and size remain unchanged
  • 4
    Direction: Positive angle = counter-clockwise
02 / Mathematics

Rotation Formula

The 2D rotation transformation uses trigonometric functions to calculate new coordinates after rotation around the origin.

Rotation Matrix

For a point (x, y) rotated by angle θ around the origin, the new coordinates (x', y') are calculated using:

x' = x·cos(θ) − y·sin(θ)
New x-coordinate
y' = x·sin(θ) + y·cos(θ)
New y-coordinate

Matrix Form

The transformation can be expressed as matrix multiplication:

[x'] [cos(θ) −sin(θ)] [x]
[y'] = [sin(θ) cos(θ)] [y]
Rotation matrix R(θ)
  • θ
    Angle in Radians: θ must be in radians (degrees × π/180)
  • !
    Origin Rotation: This formula rotates around (0, 0)

04 / Pivot Points

Rotation Around Center

Objects can rotate around different pivot points. The choice of center dramatically affects the result.

Rotation About Origin
The object orbits around the origin point. Both position and orientation change as the object moves along a circular path.
Rotation About Object Center
The object spins in place. Only orientation changes while the center position remains fixed. This is achieved by translating to origin first.

Center Rotation Formula

To rotate around an arbitrary point (cx, cy), use these steps:

1. Translate: x'' = x − cx, y'' = y − cy
2. Rotate: x''' = x''·cos(θ) − y''·sin(θ)
           y''' = x''·sin(θ) + y''·cos(θ)
3. Translate back: x' = x''' + cx, y' = y''' + cy
Three-step rotation process
  • 1
    Translate: Move object so center is at origin
  • 2
    Rotate: Apply standard rotation formula
  • 3
    Translate Back: Move object to original position
