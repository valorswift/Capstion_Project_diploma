01 / Introduction

What is a Coordinate System?

Every pixel on a screen has a unique position identified by a pair of numerical coordinates that specify its location in the display grid.

PIXEL_GRID_DEMO

Definition

A coordinate system provides a framework for identifying the position of every pixel on a display screen using numerical values:

Pixel Position = (x, y)
Coordinate representation
  • X
    X-coordinate: Horizontal position, increases from left to right
  • Y
    Y-coordinate: Vertical position, increases from top to bottom
  • O
    Origin (0,0): Located at the top-left corner of the screen

03 / Comparison

Screen vs Cartesian Coordinates

Understanding the difference between screen coordinates and mathematical Cartesian coordinates is essential for graphics programming.

Two Different Coordinate Systems

Screen Coordinate System
Cartesian Coordinate System

Screen Coordinate System

Origin: Top-Left Corner
X Right
Y Down

Used by computer displays and graphics APIs. Y values increase downward, which is opposite to the mathematical convention.

Cartesian Coordinate System

Origin: Center
X Right
Y Up

Standard mathematical coordinate system. Y values increase upward. Requires transformation when mapping to screen coordinates.

screen_y = screen_height - cartesian_y

05 / Application

Why Coordinate Systems Matter

All graphics algorithms rely on coordinate systems to position and render shapes, lines, and transformations accurately.

Graphics Operations

Coordinate systems are fundamental to virtually every graphics operation:

  • L
    Line Drawing: Algorithms use start and end coordinates to plot pixels
  • C
    Circle Drawing: Center coordinates and radius define the shape
  • P
    Polygons: Multiple vertices connected by coordinates
  • T
    Transformations: Translation, rotation, and scaling modify coordinates
