01 / Definition

What is a Pixel?

A pixel (picture element) is the smallest addressable unit of a digital display. Screens are composed of thousands or millions of pixels arranged in a grid.

PIXEL_GRID_VISUALIZATION

Definition

A pixel is a single point in a raster image. Each pixel stores color information that contributes to the overall image displayed on screen.

Pixel = (x, y) position + Color value
Basic pixel composition
  • 1
    Smallest Unit: Cannot be divided further on a display
  • 2
    Grid Arrangement: Pixels are organized in rows and columns
  • 3
    Color Storage: Each pixel holds RGB color values

03 / Resolution

Understanding Resolution

Resolution defines the number of pixels in each dimension that can be displayed. Higher resolution means more detail.

RESOLUTION_COMPARISON

Resolution Formula

Resolution is expressed as width times height, representing the total number of pixels that make up the display.

Resolution = Width x Height
Total display pixels

Common Resolutions

Standard display resolutions in modern devices:

Low
8 x 8 = 64 pixels
Medium
16 x 16 = 256 pixels
High
32 x 32 = 1,024 pixels
Ultra
64 x 64 = 4,096 pixels
04 / Color Model

Pixel Color Representation

Each pixel stores color information using the RGB color model - combining Red, Green, and Blue light intensities.

RGB Color Model

Digital displays create colors by combining different intensities of red, green, and blue light. Each channel ranges from 0 (off) to 255 (maximum intensity).

The Frame Buffer

The frame buffer is a portion of memory that holds the color values for every pixel on the display. Graphics algorithms write to this buffer to render images.

  • 1
    Memory Storage: Frame buffer stores pixel color data
  • 2
    Scan Conversion: Converting geometry to pixels
  • 3
    Refresh Rate: Display reads buffer 60+ times per second
Frame Buffer Size = Width x Height x Color Depth

06 / Next Steps

From Pixels to Algorithms

Since screens use discrete pixels, graphics algorithms must determine which pixels should be illuminated to represent shapes.

The Rasterization Challenge

Converting continuous mathematical shapes into discrete pixel representations is called rasterization. This is the fundamental problem that graphics algorithms solve.

  • 1
    Lines: Which pixels form a straight line between two points?
  • 2
    Circles: Which pixels approximate a curved boundary?
  • 3
    Polygons: Which pixels fall inside a shape?

Efficiency Matters
