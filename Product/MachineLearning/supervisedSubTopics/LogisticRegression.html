Understanding Logistic Regression

Logistic Regression is a supervised learning algorithm used for binary classification problems. Despite its name, it's used for classification rather than regression. It models the probability that an input belongs to a particular class.

The algorithm applies the sigmoid function to transform any real-valued number into a value between 0 and 1. This output represents the probability of belonging to the positive class. The decision boundary is formed where probability equals 0.5.

Training: The model learns optimal weights through gradient descent, minimizing the log loss (cross-entropy) function. This process adjusts the decision boundary to best separate the two classes.

Key Concepts
  • Sigmoid Function Maps any input to a value between 0 and 1: σ(z) = 1 / (1 + e^-z)
  • Probability Output The sigmoid output represents the probability of belonging to the positive class.
  • Decision Boundary The line where probability equals 0.5, separating the two classes.
  • Log Loss The loss function used to train logistic regression, measuring prediction error.
  • Feature Weights Parameters learned during training that determine the importance of each feature.

Advantages of Logistic Regression

Outputs well-calibrated probabilities, not just class predictions, enabling confident decision-making.
Highly interpretable model - weights directly indicate feature importance and direction.
Computationally efficient with fast training and prediction times, even on large datasets.
No strict assumptions about data distribution, works well with linearly separable classes.

Understanding the Sigmoid Function

The sigmoid function σ(z) = 1/(1+e^-z) maps any real number to a value between 0 and 1. When z = 0, the output is 0.5. Positive z values produce outputs greater than 0.5, while negative z values produce outputs less than 0.5.

Decision Boundary

The decision boundary is where the model's predicted probability equals 0.5. This occurs when z = w·x + b = 0. The boundary separates the feature space into two regions: one where the model predicts class 0, and another where it predicts class 1.

Gradient Descent Training

The model learns by iteratively adjusting weights to minimize log loss. Gradient descent computes the gradient of the loss with respect to each weight and updates them in the direction that reduces error. Learning rate controls step size.
