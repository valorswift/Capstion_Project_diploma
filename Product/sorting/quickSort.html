  3. 3. Recursively sort subarrays

Pseudocode
quickSort(arr, low, high):
if low < high:
pi = partition(arr, low, high)
quickSort(arr, low, pi - 1)
quickSort(arr, pi + 1, high)
partition(arr, low, high):
pivot = arr[high]
i = low - 1
for j = low to high - 1:
if arr[j] < pivot:
i++; swap(arr[i], arr[j])
swap(arr[i+1], arr[high])
return i + 1

About Quick Sort

Quick Sort is a highly efficient, in-place sorting algorithm using divide and conquer.

Pivot: Element used to partition the array

Partition: Rearrange elements around pivot

Recurse: Sort left and right subarrays

Quick Sort is often fastest in practice due to cache efficiency and low constant factors. However, it can degrade to O(n²) with poor pivot choices.
