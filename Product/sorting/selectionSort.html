Welcome to Selection Sort Visualizer!

Selection Sort divides the array into a sorted and unsorted region.

It repeatedly finds the minimum element from the unsorted region and moves it to the sorted region.

Pseudocode
for i = 0 to n-1:
minIndex = i
for j = i+1 to n:
if arr[j] < arr[minIndex]:
minIndex = j
if minIndex != i:
swap(arr[i], arr[minIndex])

About Selection Sort

Selection Sort is an in-place comparison sorting algorithm.

Sorted Region: Elements at the beginning that are in final order

Unsorted Region: Remaining elements to be sorted

Selection: Find minimum from unsorted region

Swap: Move minimum to end of sorted region

Selection Sort performs O(n²) comparisons in all cases, but only O(n) swaps, making it useful when writes are expensive.
