Welcome to Merge Sort Visualizer!

Merge Sort uses the Divide and Conquer technique:

  1. 1. Divide - Split array into halves
  2. 2. Conquer - Recursively sort subarrays
  3. 3. Merge - Combine sorted subarrays

right = arr[mid..end]

About Merge Sort

Merge Sort is a stable, comparison-based sorting algorithm using Divide and Conquer.

Divide: Split array into two halves

Conquer: Recursively sort each half

Merge: Combine sorted halves

Unlike Quick Sort, Merge Sort guarantees O(n log n) in all cases and is stable, making it ideal for linked lists and external sorting.
