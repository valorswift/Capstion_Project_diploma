Pseudocode
Build MaxHeap(arr)
for i = n/2-1 downto 0
heapify(arr, i)
// Heap built
for i = n-1 downto 1
swap(arr[0], arr[i])
heapify(arr, 0, i)
// Sorted!

Heap Property

In a Max Heap, each parent node is greater than or equal to its children.

For node at index i:

  • Left child: 2i + 1
  • Right child: 2i + 2
  • Parent: floor((i-1)/2)
About Heap Sort

Heap Sort is a comparison-based sorting algorithm using a binary heap data structure.

It works by:

  1. 1. Building a Max Heap from the array
  2. 2. Swapping root (max) with last element
  3. 3. Reducing heap size and heapifying
  4. 4. Repeating until sorted

Unlike Bubble Sort, Heap Sort guarantees O(n log n) in all cases, making it efficient for large datasets.
