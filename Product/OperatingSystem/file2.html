Command Reference

File Operations

mkdir [name] Create directory

Example: mkdir documents

touch [name] Create file

Example: touch readme.txt

rm [name] Delete file or empty directory

Example: rm oldfile.txt

read [file] Simulate reading a file

Shows disk seek operations and block access

Navigation

ls List directory contents
cd [path] Change directory

Examples: cd docs, cd .., cd /

pwd Print working directory
tree Show directory tree structure

Information

stat [name] Show file/directory details
df Show disk usage statistics

System Operations

defrag Defragment the disk

Reorganizes files for contiguous block allocation

format Reset and format disk

Warning: Erases all data!

clear Clear terminal
help Show this help
