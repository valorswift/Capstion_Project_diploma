What is a NOT Gate?

The NOT Gate (also called an Inverter) is the simplest digital logic gate with a single input and a single output. It performs logical negation — whatever the input is, the output becomes the opposite.

Real-World Applications: Signal inversion in communication circuits, creating complementary clock signals in microprocessors, building oscillators when combined with a resistor and capacitor, and as a fundamental building block in all complex logic designs.

Boolean Algebra: Q = A'

Advanced Theory

Double Negation

Applying NOT twice returns the original value — a fundamental identity in Boolean algebra:

A'' = A

Two inverters in series act as a non-inverting buffer, often used to strengthen a weak signal without changing its logic level.

Transistor-Level Implementation

A NOT gate requires only one transistor in CMOS: a PMOS pulls the output HIGH when input is LOW, and an NMOS pulls it LOW when input is HIGH. This makes it the most efficient gate to fabricate.

Propagation Delay

The 7404 TTL inverter has a typical propagation delay of ~10 nanoseconds. CMOS versions like the 74HC04 achieve ~8ns with significantly lower static power consumption.
