What is an XNOR Gate?

The XNOR Gate (Exclusive-NOR) is the logical complement of the XOR gate. It outputs HIGH only when both inputs are identical — either both LOW or both HIGH. For this reason, it is also called the Equivalence Gate or Coincidence Gate.

Real-World Applications: Digital comparators for checking if two binary numbers are equal, pattern matching circuits in digital signal processing, error-checking in communication systems, and address decoding in memory circuits.

Standard Notation: Y = (A ⊕ B)'
Equivalence: Y = A ⊙ B
SOP Form: Y = A'B' + AB

Advanced Theory

Equality / Coincidence Gate

The XNOR gate is the only basic gate that detects exact equality between inputs. This makes it the fundamental building block of digital comparators used in ALUs to determine if two binary values match.

Complement Relationship

XNOR is the exact logical inverse of XOR. This means:

XNOR = NOT(XOR) = XOR'

In hardware, an XNOR gate is often built by connecting a 7486 XOR output directly into a 7404 NOT gate, rather than using a dedicated XNOR IC.

Multi-Bit Comparator

To compare two n-bit numbers, XNOR each pair of bits individually, then AND all results together:

Eq = (A0⊙B0) · (A1⊙B1) · ... · (An⊙Bn)

Result is HIGH only when every bit matches.

Controlled Buffer

Scenario: Tie one input to a control signal.
Control = 0 → Y = NOT(A) (inverting buffer)
Control = 1 → Y = A (non-inverting buffer)

This is the opposite behavior of the XOR controlled inverter — useful in bus transceiver circuits.
