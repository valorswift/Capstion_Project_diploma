Algorithm Code
1 while low <= high
2 mid1 = low + (high-low)/3
3 mid2 = high - (high-low)/3
4 if arr[mid1] == target
5 return mid1
6 if arr[mid2] == target
7 return mid2
8 if target < arr[mid1]
9 high = mid1 - 1
10 else if target > arr[mid2]
11 low = mid2 + 1
12 else
13 low = mid1 + 1
14 high = mid2 - 1
15 return -1

Welcome to the Ternary Search Visualizer!

Ternary Search works on sorted arrays by dividing the search range into three parts using two midpoints.

Key Formulas:

mid1 = low + (high - low) / 3

mid2 = high - (high - low) / 3
