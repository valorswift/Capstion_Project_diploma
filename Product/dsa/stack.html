Stack Basics

A linear data structure following Last-In-First-Out (LIFO) principle.

What is a Stack?

A stack is a linear data structure that follows the Last-In-First-Out (LIFO) principle. The last element inserted is the first one to be removed. Think of it like a stack of plates - you can only add or remove from the top.

Key Concept

Only the top element is accessible at any time. You cannot access elements below without removing the ones above.

Implementation

Can be implemented using arrays (fixed size) or linked lists (dynamic size). Each has its own trade-offs.

Stack Operations

Core operations that define stack behavior.

Push

Add an element to the top of the stack. Time complexity: O(1)

stack.push(element)

Pop

Remove and return the top element. Time complexity: O(1)

element = stack.pop()

Peek / Top

Return the top element without removing it. Time complexity: O(1)

element = stack.peek()

isEmpty

Check if the stack is empty. Time complexity: O(1)

if (stack.isEmpty()) {...}

Expression Evaluation

How stacks are used to evaluate arithmetic expressions with proper operator precedence.

The Two-Stack Algorithm

Dijkstra's two-stack algorithm uses one stack for operands and another for operators. It processes each token and applies operators based on precedence rules.

Supported Operators

+ Addition
- Subtraction
* Multiply
/ Division
^ Power
( ) Parens

Operator Precedence

Higher precedence operators are evaluated first. When operators have equal precedence, associativity determines the order (left-to-right for most, right-to-left for ^).

Real-World Applications

Stacks are fundamental in computer science with countless applications.

Function Calls

Call stack manages function invocation and return addresses in programming languages.

Undo Operations

Text editors use stacks to track changes and implement undo functionality.

Browser History

Back/forward navigation uses stacks to track visited pages.

Syntax Parsing

Compilers use stacks to check balanced parentheses and parse expressions.

Memory Management

Stack memory allocation for local variables and function parameters.

DFS Algorithm

Depth-first search uses stacks (explicit or call stack) to traverse graphs.
