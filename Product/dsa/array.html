Array Basics

A collection of items stored at contiguous memory locations.

What is an Array?

An array is a data structure consisting of a collection of elements, each identified by at least one array index or key. Unlike linked lists, arrays store data in a contiguous block of memory. This allows for extremely fast access using simple math.

Static Arrays

Fixed size allocated at compile time. Memory is reserved upfront. Fast, but size cannot change.

Dynamic Arrays

Size can grow. When full, a larger block is allocated, data is copied, and old block is freed (e.g., ArrayList in Java, std::vector in C++).

Memory Representation

How arrays map to physical RAM addresses.

Address Calculation Formula

Because memory is contiguous, we can calculate the address of any element instantly using:

Address(A[i]) = Base_Address + (i × Element_Size)

Common Operations

Traversal

Visiting every element exactly once. Used for printing, searching, or aggregating.

Insertion

Adding an element. In static arrays, this requires shifting all subsequent elements to make space.

Deletion

Removing an element. Requires shifting the trailing elements left to fill the gap.

Sorting

Arranging elements in a specific order (Asc/Desc). Algorithms include Bubble, Quick, and Merge Sort.

1D vs 2D Arrays

1D Array (Linear)

A single row of elements. Memory addresses are sequential.

int arr[5] = {1, 2, 3, 4, 5};

2D Array (Matrix)

Can be thought of as an array of arrays. Stored linearly in memory using Row-Major or Column-Major order.

int matrix[2][3] = {{1,2,3}, {4,5,6}};

Row-Major Order Addressing

In most languages (C, C++, Java), 2D arrays are stored row by row.

Address(arr[i][j]) = Base + ((i * cols) + j) * size

Arrays in Microprocessors

How hardware handles array data.

Hardware Data Flow

  • Base Address Register: CPU holds the starting address of the array.
  • Indexed Addressing: CPU calculates physical address using Base + (Index * Offset) in one cycle.
  • Cache Locality: Because arrays are contiguous, reading one element loads the next few into the CPU cache automatically.

Real-World Applications

Image Processing

Images are 2D arrays (matrices) of pixels. Each pixel contains color values (RGB).

Databases

While DBs use B-Trees, the underlying buffer pool stores data pages in arrays.

Game Dev

Used for inventory systems, tile maps, and entity lists for performance.

Stacks & Queues

These abstract data types are typically implemented using arrays for efficiency.
